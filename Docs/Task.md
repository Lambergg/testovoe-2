## Задача: 
- Реализовать систему авторизации и аутентификации с использованием Redis, 
для реализации черного и белого списка токенов (JWT).

### Дополнительные задачи: \
- выбрать способ взаимодействия с контентом (объяснить свой выбор); \
- определить возможные утечки токенов, как определить, что токен украден и способы защиты; \
- реализовать систему ролей, например, 2е роли имеют один общий контент и один, который принадлежит только этой роли; \
- упаковать данное решение в контейнер.  

P.S. Клиентской части уделить как можно меньше внимания, продемонстрируйте знания и умения Backend части.
